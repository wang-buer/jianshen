# 基于springboot的健身房会员管理系统4



## 1、项目介绍

基于springboot的健身房会员管理系统拥有多种角色

用户：登录注册、购买健身课程、兑换商品、评价收藏等

管理员：用户教练管理、课程管理、商品管理、基础数据管理、公告管理等

教练：回复帖子、查看各种信息

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：前端html、后端VUE前后分离

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
